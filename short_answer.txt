3) Explain the difference between these two commands in the terminal:
   
   $ cd Users
   $ cd /Users

Your answer:

The first command lets you go in the directory Users from the directory you're currently in (if Users directory exists in the directory you're currently in). The second command changes the path to the Users directory within the root directory of the computer (if Users directory exists within the root directory of the computer).

8) What is the difference between `git add` and `git commit`?

Your answer:
git add adds the files to be staged (tells git to start tracking the files). git commit takes the staged changes and takes a snapshot of them (creates a snapshot of a current version of a code that is possible to retrieve later)
